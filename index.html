Ventas Chatbot

La aplicación documentada corre con Flask en la carpeta backend_python/.

  1. Configure un archivo .env en la raíz del repositorio (variables de BD, GROQ_API_KEY, FLASK_SECRET; puede usar api/python/.env.example como guía de nombres).
  2. Instale dependencias: cd backend_python y pip install -r requirements.txt.
  3. Arranque: python -m flask --app app run --host 127.0.0.1 --port 5000.
  4. Abra http://127.0.0.1:5000/ o http://127.0.0.1:5000/login (también acepta /login.php por compatibilidad).

En api/python/ hay una variante FastAPI; el flujo principal del repo es el de Flask anterior.
